# Roadmap de desenvolvimento do Game Shelf

## Ordem aprovada

1. Fase 0: fundacao do repositorio
2. Fase 1: MVP persistido
3. Fase 2: qualidade do dominio e usabilidade
4. Fase 3: integracao Steam MVP
5. Fase 4: progresso rico
6. Fase 5: analytics e organizacao avancada
7. Fase 6: expansao e ecossistema

## Regra de decisao

Antes de criar uma feature, perguntar:

- isso resolve dor real do usuario?
- isso pertence ao nucleo ou ao apoio?
- isso cabe na fase atual?
- qual o menor corte vertical util?

Se a resposta indicar duvida ou dependencia futura, manter escopo menor.
